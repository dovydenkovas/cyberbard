# Cyberbard

[![Build](https://github.com/dovydenkovas/cyberbard/actions/workflows/build.yml/badge.svg)](https://github.com/dovydenkovas/cyberbard/actions/workflows/build.yml)

## Description

Cyberbard is a music player for tabletop role-playing games and theatrical improvisations.

Main features:
- [x] Support mp3, ogg, wav and flac music.
- [x] Add any tags for music.
- [x] Playlists.
- [x] A graphical map with locations. Each location could contain playlists and other locations.
- [x] Threading playlists. You can play music and effects at one time.

## Installation
### Binary
Go to [releases page](https://github.com/dovydenkovas/cyberbard/releases) and download the archive with Cyberbard for your system.

### Build from source
Requires stable rust with cargo.
Download or clone sources, then run in project's root directory:

```sh
cargo build --release
```

Artifacts will be in `target/release` directory.
